#30 VORISLIK VA POLIMORFIZM
Klasslardan klass yaratishni o'rganamiz.
SUPER KLASS VA VORIS KLASS
Obyektga yo'naltirilgan dasturlashning qulayliklaridan biri bu klasslardan boshqa klass yaratish imkoniyati.
